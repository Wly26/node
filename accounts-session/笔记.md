这是一个基于
mongodb数据库的
nodejs中的
express框架下的
generator插件的
快速开发项目。
### 接入接口，前后端开始分离

安装--------------------------------

1，全局配置：
2，npm init
3，npm i express
4，npm install -g express-generator
5，express -e accounts
6，cd accounts

7，局部配置：
8，npm install -g nodemon
9，改package.json 
  "scripts": {
    "start": "nodemon ./bin/www"
  },
10，npm i
11，npm start


另外安装------------------------------
mongoose
md5
express-session
connect-mongo

------------------------------------

本机
